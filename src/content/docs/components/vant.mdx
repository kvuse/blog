## 介绍

- `vant`二次封装组件
- 组件快速使用，避免重复开发, 方便项目开发使用

## 安装

```bash
# Npm  
npm install @kvuse/components
 
# Yarn  
yarn add @kvuse/components

# Pnpm  
pnpm add @kvuse/components
```

## 使用

### 全局引入

```js
// main.js
import { createApp } from 'vue';
import { KVant } from '@kvuse/vant';
import '@kvuse/vant/dist/index.css';

Vue.use(KVant)
```

### 按需引入

```js
import {
  KVant,
  KvButton,
  KvDatePicker,
  KvInput,
  KvList,
  KvStatus,
  KvTable,
  KvTree
} from '@kvuse/vant';

export default {
  components: {
    KVant,
    KvButton,
    KvDatePicker,
    KvInput,
    KvList,
    KvStatus,
    KvTable,
    KvTree  
  },
};
```
